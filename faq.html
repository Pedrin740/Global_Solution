Perguntas Frequentes

O que é o SafeRoute?

Plataforma inteligente de segurança urbana.

Como registrar uma ocorrência?

Após o login, informe localização e detalhes.

Quem pode cadastrar ocorrências?

Qualquer usuário cadastrado.

Como funciona o nível de risco?

O sistema analisa quantidade e frequência das ocorrências.

Quais tipos de ocorrências podem ser registradas?

Furtos, assaltos, iluminação inadequada, atividades suspeitas e outras situações que possam representar risco à população.

Como os alertas de segurança funcionam?

O sistema envia notificações quando identifica aumento no número de ocorrências ou quando o usuário está próximo de uma área considerada perigosa.

As informações registradas são verificadas?

Sim. As ocorrências passam por processos de validação para reduzir informações falsas, incorretas ou duplicadas.

O que é o Dashboard Interativo?

É uma área da plataforma que apresenta mapas, gráficos e estatísticas sobre a segurança urbana de forma visual e fácil de entender.

Como o sistema sugere rotas mais seguras?

O SafeRoute considera os níveis de risco das regiões e as ocorrências registradas para recomendar trajetos com maior segurança.

O chatbot pode responder dúvidas sobre segurança?

Sim. O chatbot utiliza os dados da plataforma para responder perguntas sobre regiões, rotas, horários e níveis de risco.

Meus dados pessoais ficam protegidos?

Sim. A plataforma utiliza mecanismos de controle de acesso e proteção das APIs para garantir a segurança das informações dos usuários.

Posso acessar o sistema pelo celular?

Sim. O SafeRoute foi desenvolvido para funcionar em diferentes dispositivos, incluindo computadores, tablets e smartphones.
